# Easy Shop Mobile 📱

Landing page responsiva para um app de compras mobile, desenvolvida como desafio prático do programa **DevClub**.

---

## Demonstração

> Abra o arquivo `index.html` diretamente no navegador ou utilize uma extensão como **Live Server** no VS Code.

---

## Tecnologias

- **HTML5** — estrutura semântica da página
- **CSS3** — estilização, Flexbox e media queries
- **JavaScript** (Vanilla) — interatividade do menu mobile
- **Google Fonts** — tipografia Oswald

---

## Funcionalidades

- Layout responsivo para desktop e mobile
- Menu de navegação com hamburger animado no mobile
- Tipografia fluida com `clamp()` (se adapta ao tamanho da tela)
- Container centralizado com `max-width: 1440px`
- Altura máxima de 812px no mobile

---

## Breakpoints

| Breakpoint | Comportamento |
|---|---|
| `> 1440px` | Conteúdo centralizado na tela (ultrawide) |
| `≤ 768px` | Layout empilhado, hamburger ativado |
